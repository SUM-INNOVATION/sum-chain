The protocol 01

Storage

Files live on the protocol itself. Archive nodes stake Koppa, hold encrypted data, and answer per-block retrievability challenges with Merkle proofs recorded on chain.

Per-block PoR challenges · 5% slash on failed proof → 02

Compute

OmniNode settles verifier-signed attestations of AI inference on chain. Pay once for compute; the proof that it ran stays verifiable forever.

One attestation per session and verifier, permanent → 03

Governance

Policy accounts put multi-signature group control in the state executor, not in smart contracts. Thresholds per action class, fail-closed by default.

Enforced at consensus level · epochs of 14,400 blocks → 04

Tokenomics

800 billion Koppa, all minted at genesis. No inflation, no block rewards. Fees are 0.001 Ϙ flat and go to the validator who proposes the block.

800B Ϙ fixed supply · zero inflation → 05

Run a node

The node is one Rust binary built from a single command. Three-second blocks, deterministic finality, and a JSON-RPC surface for monitoring.

Rust 1.85 · 3s blocks · ~18s finality → 06

Wallet

SUMailet runs in the browser with nothing to install. The CLI wallet lives in the terminal. Both are non-custodial and talk to the same chain.

Web and CLI · fees from 0.001 Ϙ →
